# M5_Encoder

## Overview

Simple library for the [Encoder Unit | m5stack-store](https://shop.m5stack.com/products/encoder-unit).

## Description

This is a library for the M5Stack Unit Encoder (SKU: U135). It allows you to easily interface with the encoder and read its position and button state or set its pixels.

It is based on [m5stack/M5Unit-Encoder: Contains case programs of M5Stack Unit Encoder.](https://github.com/m5stack/M5Unit-Encoder)
